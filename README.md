# Alexandria

A reading platform and personal library for articles, books, podcasts, and videos.

Word-by-word text-to-speech with synced highlights. Built to help you understand and remember what you read.

[**alexandria.live**](https://alexandria.live) — free Chrome extension and web app.

## What it is

Save anything: articles from the web, books as EPUB, podcasts, YouTube videos, even your inbox. Alexandria reads it back word-by-word with the current word highlighted in sync with the audio. Works on desktop via the Chrome extension and as a Progressive Web App on iOS, Android, and the web.
